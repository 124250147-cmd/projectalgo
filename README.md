Sistem Pemesanan Restoran

Project ini merupakan program berbasis C++ untuk mengelola pemesanan di restoran dengan menggunakan konsep Algoritma dan Struktur Data.

Fitur program yang kami gunakan adalah:
- Menampilkan daftar menu
- Mencari menu (Searching)
- Mengurutkan menu berdasarkan harga (Sorting)
- Menambah pesanan
- Menghapus pesanan
- Menampilkan pesanan
- Checkout transaksi
- Menyimpan transaksi ke file

Materi yang kami gunakan dalam project ini adalah: Digunakan
1. Pointer  
Digunakan untuk membuat node pada linked list.
2. Linked List & Double Linked List  
Digunakan untuk menyimpan data pesanan sementara.
3. Searching (Sequential Search)  
Digunakan untuk mencari menu berdasarkan kode.
4. Sorting (Bubble Sort)  
Digunakan untuk mengurutkan menu berdasarkan harga.
5. Operasi File  
Digunakan untuk menyimpan data transaksi ke file transaksi.txt
6. Database (SQL)  
Struktur database disediakan dalam file database.sql
